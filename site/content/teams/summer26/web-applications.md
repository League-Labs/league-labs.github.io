title: Web Applications

The Summer 2026 Web Applications team builds the web tooling that supports League Labs projects and communication.

Areas of work:

* small internal tools for signups, schedules, or coordination
* dashboards and project status pages
* documentation sites and team resources
* frontend and backend workflows for student-built apps

Suggested outputs:

* a working web app or prototype
* deployment notes and contributor instructions
* reusable UI or content patterns for future cohorts
